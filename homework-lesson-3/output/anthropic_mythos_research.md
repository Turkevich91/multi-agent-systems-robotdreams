# Дослідження моделі Anthropic Claude Mythos (Capybara)

## Короткий висновок
**Claude Mythos** (також відома під назвою **Capybara**) — це нова, надпотужна модель штучного інтелекту від компанії Anthropic, яка представляє собою "якісний стрибок" (step change) у можливостях ШІ. Модель демонструє безпрецедентні результати в задачах програмування, академічного мислення та, особливо, у сфері кібербезпеки. Її існування стало відомим після випадкового витоку даних, що призвів до публікації чернеток офіційних анонсів компанії.

## Аналіз можливостей
На основі доступних даних, Mythos суттєво перевершує попередню флагманську модель Claude Opus 4.6 за ключовими показниками:

*   **Кібербезпека (Cybersecurity):** Це найсильніша сторона моделі. Вона здатна самостійно знаходити та експлуатувати вразливості "нульового дня" (zero-day) у популярних операційних системах (Linux, FreeBSD, OpenBSD) та веббраузерах. 
    *   *Приклад:* Якщо Opus 4.6 мав майже нульовий успіх у розробці експлойтів, то Mythos змогла створити робочі експлойти 181 раз із сотень спроб.
    *   Модель здатна створювати складні ланцюжки вразливостей (exploit chains), обходити пісочницю (sandboxing) та здійснювати підвищення привілеїв (privilege escalation).
*   **Програмування та логіка:** Модель демонструє значно вищі бали в тестах на написання коду та складне академічне мислення порівняно з лінійкою Opus.
*   **Рівень інтелекту:** Внутрішньо Mythos (або Capybara) позиціонується як новий, ще більший і розумніший рівень моделей, що стоїть вище за серію Opus.

## Практичні trade-offs (Компроміси)
Використання такої моделі несе в собі як величезні можливості, так і критичні ризики:

| **Переваги** | **Ризики та обмеження** |
| :--- | :--- |
| **Прорив у безпеці:** Можливість автоматичного виявлення вразливостей для їх швидкого виправлення (Project Glasswing). | **Кіберзагроза:** Ризик використання моделі хакерами для створення автоматизованих атак на критичну інфраструктуру. |
| **Підвищення продуктивності:** Революційні можливості в кодуванні та складних наукових розрахунках. | **Висока вартість:** Модель є надзвичайно дорогою в експлуатації, що робить її доступною лише для обмеженого кола клієнтів. |
| **Автоматизація захисту:** Створення інструментів для "автономного захисника". | **Доступність:** Наразі модель перебуває у стадії Private Preview (закритого тестування) через загрози безпеці. |

## Обмеження дослідження
*   **Статус моделі:** Модель ще не випущена для загального користування; більшість інформації походить з витоків та попередніх анонсів Anthropic.
*   **Відсутність публічних бенчмарків:** Оскільки модель перебуває у закритому доступі, повний набір офіційних технічних метрик (на кшталт MMLU тощо) поки що недоступний для незалежної перевірки.
